NOTE: No additional Node packages are required for this script.

1) From the command line, navigate to the directory that contains the index.js file.

2) Run the index.js file, including arguments for the web site URL, the search word, and the replacement word.

	EXAMPLE: >node index.js www.southerncompany.com Southern Northern

3) The script will retun results and the updated web page will be saved in the same directory as the index.js file.
